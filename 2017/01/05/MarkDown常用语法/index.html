很简单一个tab或者四个空格就建立了代码区块就是这么简单 再试一下 就是四个空格的距离，这个代码区会一直持续到没有缩进的那一行或者文件末尾在代码区里 &<>会自动转换成HTML实体，复制源码加上缩进就可以了

##分隔线可以使用星号三个或者三个以上*减号-底线_来创建分割线

区段元素1. 链接连接分为行内样式和参考样式

不管哪一种 连接文字都是用方括号来标记* 行内样式 行内 方括号紧接着圆括号

如果有标题用双引号括起来 如果在同一个主机资源 也可以使用相对路径 本页中的其他路劲* 参考样式 由两个方括号做成 第二个方括号填入辨识连接的标记[例子][0]在任意位置把整个标记制定的内容定义出来

[0]: http://www.baidu.com (可选标题)方括号前要不要空格都可以，方括号紧跟着冒号 空格或者tab后接网址 标题可选还可以换行 单引号双引号括号都可以

凡是标记符都应该与其他的东西隔开 最好不要写在一起
